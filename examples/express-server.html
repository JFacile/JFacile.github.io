Example of a basic express server with JFacile


1 ) Launch the server 

          
              const JFacile = require("jfacile");
              

              JFacile.run(3000);


2 ) Add the main route 

          
              const JFacile = require("jfacile");
              

              JFacile.routeGet("/get", "Get Route", "get");
              JFacile.routePost("/post", "Post Route", "post");
              JFacile.routePut("/put", "Put Route", "put");
              JFacile.routeDelete("/delete", "Delete Route", "delete");
              JFacile.run(3000);
