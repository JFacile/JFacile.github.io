Example of anti-crash process with JFacile


1 ) Basic anti-crash 

          
              const JFacile = require("jfacile");
              

              JFacile.JExit()


2 ) Advanced anti-crash 

          
              const JFacile = require("jfacile");
              

              JFacile.JExit();
              JFacile.JUnhandledRejection();
              JFacile.JUnCaughtException();
              JFacile.JWarning();
              JFacile.JRejectionHandled();
              JFacile.JuncaughtExceptionMonitor();
              JFacile.JmultipleResolves();
