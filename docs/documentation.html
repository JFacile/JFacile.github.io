Get Started


To use JFacile, you do install the package in your project.

$ npm install jfacile



Good ! Now, you can require the module in your file. 

          
              const JFacile = require("jfacile");


Now, you can use JFacile.

Express Server


JFacile is powerful to create simple express server.
You can create a express server and a route in 3 lignes of code with JFacile.

run(port) : port = number

routeGet(route, titleOfThePage, render) :
  • route = string
  • titleOfThePage = string
  • render = string

routePost(route, titleOfThePage, render) :
  • route = string
  • titleOfThePage = string
  • render = string

routePut(route, titleOfThePage, render) :
  • route = string
  • titleOfThePage = string
  • render = string

routeDelete(route, titleOfThePage, render) :
  • route = string
  • titleOfThePage = string
  • render = string
